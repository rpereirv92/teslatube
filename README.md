# TeslaTube V2++

🚀 Projet Master – UX multimédia Tesla

## Installation
1. Copier le dossier complet `teslatube-v2plus++` sur votre serveur web
2. Remplacer `js/app.js` avec votre clé YouTube API
3. Ouvrir `index.html` depuis navigateur Tesla
4. Utiliser `phone.html` pour contrôle passager et ajout favoris

## Fonctionnalités
- UI Netflix ultra réaliste
- Animations smooth, hover preview
- Intégration YouTube officielle
- Mode audio intelligent en conduite
- Favoris synchronisés (localStorage)
- Recommandations simples basées sur historique
- Télécommande passager
